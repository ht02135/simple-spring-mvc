1>$ docker pull <image_name>

2>$ docker ps [options..]
    -a flag:  shows us all the containers, stopped or running.
    -l flag: shows us the latest container.
    -q flag: shows only the Id of the containers. 

3>$ docker run <image_name>
To give name of container
$ docker run --name <container_name> <image_name>

4>$ docker stop <container_ID>

5>$ docker start <container_ID>

To delete a container
6>$ docker rm {options} <container_name or ID>

7>To delete the image in docker. You can delete the images which are useless from the docker local storage so you can free up the space

docker rmi <image ID/ image name>

8>Lists all the pulled images which are present in our system.

$ docker images

9>The docker build command is used to build the docker images with the help of Dockerfile.

docker build -t image_name:tag .

10>While running the containers in Docker you may face some errors and containers fails to start. You can restart the containers to resolve the containers by using the following commands.

docker restart container_name_or_id
